# Денисюк Прожект Ремба

Одностраничный статичный birthday landing по Figma-наброску.

## Локальный запуск

Откройте `index.html` в браузере или поднимите любой статичный сервер в этой папке.

## Запуск внутри локальной сети

В PowerShell из папки проекта:

```powershell
.\start-lan.ps1
```

Сейчас сайт можно открыть с устройств в той же сети по адресу:

```text
http://192.168.1.26:4173/index.html
```

Если Windows Firewall заблокирует доступ, разрешите входящие подключения для Python или добавьте правило для TCP-порта `4173` от имени администратора.

## Деплой

Проект не требует сборки. Его можно загрузить на GitHub Pages, Vercel как static site или любой другой бесплатный хостинг.
